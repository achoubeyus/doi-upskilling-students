# About the Course
This course offers a series of technical presentations and interactive discussions, complemented by hands-on Jupyter Notebooks, to help you build and deploy solutions using AI Foundry. You’ll gain practical experience creating Foundry projects, deploying models and AI services (including AI Search), exploring capabilities through the Chat Playground, running code from Visual Studio, deploying agents, and applying Foundry to real-world use cases.
